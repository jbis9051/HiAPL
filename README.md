## Where?

Client side and server side. For node, simply look at the import syntax:

```html
<head>
    <meta type="import" module="fs">
</head>
<!-- Now You can use the module -->
<text type="String">
  <fs>
    <readFileSync>./text.txt</readFileSync>
  </fs>
</text>
```
